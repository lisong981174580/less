> http://lesscss.cn
#### 编译软件
> http://koala-app.com/  //考拉
#### 命令行编译
> 全局安装less插件
```
npm install  -g lessc
lessc index.less  index.css

```
#### 外部库
> less.min.js可以在官网下载
> 注意在服务器的状态下启动
```
<link  rel="stylesheet/less"  type="text/css"  href="less/index.less">
<script  type="text/javascript" src="js/less.min.js><script>

```
